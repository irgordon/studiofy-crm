# Studiofy CRM (Elementor Edition)

**Version:** 2.0.0  
**Requires PHP:** 8.1+  
**Requires WordPress:** 6.6+  
**Requires Elementor:** 3.0+  
**Author:** Ian R. Gordon

## Overview
Studiofy CRM is a professional business management suite for photographers. Version 2.0 introduces native **Elementor Widgets**, allowing you to build Client Portals, Lead Forms, and Booking Pages visually using the Elementor page builder.

## Key Features
1.  **Elementor Widgets:**
    * **Studiofy Lead Form:** Drag-and-drop inquiry form linked to the CRM.
    * **Studiofy Gallery:** Visual proofing grids with watermarking support.
    * **Studiofy Scheduler:** Frontend booking calendar with availability checks.
2.  **Core CRM:**
    * **Kanban Projects:** Manage workflows (To Do, In Progress, Future).
    * **Contracts:** Digital eSignatures and PDF generation.
    * **Invoicing:** Square API integration and iCal export.
    * **Client Management:** Centralized database for leads and clients.

## Installation
1.  Upload the `studiofy-crm` folder to `/wp-content/plugins/`.
2.  Activate via the WordPress Plugins menu.
3.  **Important:** Ensure Elementor is installed and active.
4.  Go to **Studiofy CRM > Settings** to configure branding and API keys.
5.  Open any page in **Elementor** and look for the "Studiofy CRM" widget category.
