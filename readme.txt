Studiofy "Everest" transforms WordPress into a native photography studio command center.

== Description ==

**Stop renting your business process.** Studiofy CRM v3.0.0 "Everest" is a comprehensive business operating system designed for high-volume photography studios. It replaces disjointed SaaS subscriptions with a unified, native WordPress dashboard.

**Everest Features:**
* **Kanban Production Board:** Visualize your workflow from Pre-Production to Delivery with drag-and-drop cards.
* **Dynamic Contracts:** Create legal templates with variable data and collect e-signatures.
* **Native Financials:** Generate Invoices linked directly to Square Payments without heavy SDKs.
* **Client Galleries:** Deliver assets securely using the native WordPress Media Library.
* **Data Seeder:** One-click demo content generation for instant setup testing.

== Installation ==

1. Upload `studiofy-crm` to `/wp-content/plugins/`.
2. Activate via WordPress Admin.
3. Navigate to **Studiofy > Settings** to configure API keys or install Demo Data.
