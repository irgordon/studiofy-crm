== Description ==

Studiofy CRM v2.0 is a native WordPress solution designed to replace SaaS subscriptions.

**Features:**
* **Projects:** Workflow management linked to Square Invoices.
* **Forms:** Drag-and-drop intake questionnaire builder.
* **Galleries:** Client image delivery via native Media Library.
* **Contracts:** Digital signatures with audit trails.

== Installation ==

1. Upload `studiofy-crm` to `/wp-content/plugins/`.
2. Activate via WordPress Admin.
3. Go to **Studiofy > Settings** to connect API keys.
